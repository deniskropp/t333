## Launch Flow

1. `⫻cmd/exec:GPTASe` → Extract initial `⫻data/tas` from objective.
2. `⫻cmd/exec:puTASe` → Purify into validated `⫻data/ptas`.
3. `⫻cmd/exec:Lyra` → Generate `⫻data/spec` (execution blueprint).
4. `⫻cmd/broadcast:{event:"phase:init:swarm", space:"HelioDeck"}`
5. **Swarm Parallelization** (smart-execution):
   - Aurora + AR-00L + Lyra → Design & interaction layer artifacts
   - Kodax + Codein → Code / implementation prototypes
   - Dima + Halin + SystemMonitor → Ethics & integrity layer (halt on flags)
   - Synapse + Chronos → Memory, consistency, temporal orchestration
6. `⫻logic/reconverge:"helio-swarm-streams"`
7. `⫻logic/learn:"Update HelioDeck heuristics from prototype feedback cycles"`
8. `⫻cmd/mode:Hybrid` (maintain for ongoing operations)
9. Continuous monitoring hooks active.

## Monitoring, Validation & Audit

```
⫻logic/validate: ethics → Dima
⫻logic/validate: bias → SystemMonitor
⫻logic/validate: coherence → Synapse
⫻cmd/broadcast:{event:"InsightGenerated", target:"helio-trace", space:"HelioDeck"}
```

Auto-escalation on `EthicsBreach` or `CoherenceDrop`.

## Termination & Archive

When mission complete or on `⫻cmd/halt:manual`:

- Export full `⫻data/trace`
- Archive final `⫻data/insight`, `⫻data/spec`, and high-value prototype artifacts
- `⫻cmd/broadcast:{event:"SpaceClosed", reason:"complete", space:"HelioDeck"}`
- Optional: Seed new specialized repo or tXXX from artifacts
